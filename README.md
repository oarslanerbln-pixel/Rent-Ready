# GlobalPulse | Startup Guide

## 🚀 Hızlı Başlangıç (Quick Start)

Bu proje **Vite + React** kullanılarak oluşturuldu. Beyaz ekran görüyorsan, bağımlılıklar henüz yüklenmemiş veya proje bir sunucu üzerinden çalıştırılmıyor demektir.

### Adım 1: Bağımlılıkları Yükle
Terminali açın ve `GlobalPulse` klasörünün içindeyken şu komutu çalıştırın:
```bash
npm install
```

### Adım 2: Geliştirme Sunucusunu Başlat
Bağımlılıklar yüklendikten sonra şu komutu çalıştırın:
```bash
npm run dev
```
Terminalde bir URL görünecektir (örneğin: `http://localhost:5173`). Bu URL'yi tarayıcınızda açın.

---

## 🛠️ Proje Yapısı
- `src/App.jsx`: Ana arayüz ve dashboard.
- `src/intelligence.js`: Gemini 2.0 Flash entegrasyonu için hazırlanan veri işleme mantığı.
- `src/index.css`: Projeye özel tasarım sistemi ve yardımcı sınıflar.

## 🎯 Hedeflenen Özellikler
1. **Pulse Map**: Kriz bölgelerinin anlık parladığı interaktif harita.
2. **Gemini Intel**: Haber akışının Gemini Flash ile anlık analizi ve güven skoru.
3. **Mobile-First**: Tamamen mobil cihazlar için optimize edilmiş akış.
